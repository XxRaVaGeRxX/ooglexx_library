# ooglexx_library
A ready-to-rock `.lib` file for c++ developers to simplify typewriter effects in their console apps. 

## INFO
> ooglexx is a simple add on to speed up development;
> 
> More Coming Soon!;

## Instructions for Use
>Download the zip containing the `.lib` & `.h`;
>
>create a header file, `ooglexx_texttool.h` in your project;
>
>Link to your project in whichever ide you use;

## Current Functions For Use {All Are Typewriter style, Except Load Bar}
>`b_type` = 40ms delay; 
>
>`q_type` = 20ms delay;
>
>`e_type` = 80ms delay, Red text For Errors;
>
>`no_newl` = 40ms delay, No New Line;
>
#### FALSE LOAD BARS
>
>`falseloadf`= 50ms delay;
>
>`falseload` = 100ms delay;
