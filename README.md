# ooglexx_library v0.1.1
A ready-to-rock `.lib` file for c++ developers to simplify typewriter effects in their console apps. 

## INFO
> ooglexx is a simple add on to speed up development;
>
> The pre-made ooglexx header for your project can be found in `ooglexx_texttool.zip` or [Files_other](https://github.com/XxRaVaGeRxX/ooglexx_library/tree/master/Files_other)
>
> The source code can be found in [Files_source](https://github.com/XxRaVaGeRxX/ooglexx_library/tree/master/Files_source)
>
> A `.txt` info file is included with the zipped files for reference of function names, etc
> 
> More Coming Soon!;

## Instructions for Use
>Download the zip containing the `.lib` & `.h`;
>
>create a header file, `ooglexx_texttool.h` in your project;
>
>Link to your project in whichever ide you use;

## Current Functions For Use {All Are Typewriter style, Except Load Bar}
>`b_type` = 40ms delay;
>
>`s_Type` = 60ms delay;
>
>`c_Type` = 80ms delay, Orange text for cautionary prompts, etc;
>
>`g_Type` = 40ms delay, Green text for positive prompts;
>
>`q_type` = 20ms delay;
>
>`e_type` = 80ms delay, Red text For Errors;
>
>`no_newl` = 40ms delay, No New Line;
>
#### FALSE LOAD BARS
>
>`falseloadf`= 50ms delay;
>
>`falseload` = 100ms delay;
>
#### SPECIAL STUFF
>`writeFile` = write a file using writeFile("example.txt","This is a new text file for my app")
